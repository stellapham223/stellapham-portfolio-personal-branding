Case Study 04

Competitor Stalker — SaaS Giám sát Đối thủ

Xây dựng nền tảng SaaS full-stack theo dõi thay đổi của đối thủ cạnh tranh trên Shopify App Store — từ ý tưởng đến production chỉ với một developer duy nhất và quy trình Agentic AI với Claude Code.

📌 Agentic AI + Full-Stack SaaS 🏢 Personal Project 🛠️ Next.js · Firebase · Puppeteer · Claude Code 🚀 Production — Phase 2 in progress

🔍 Context & Problem

Strategic Context

Trong thị trường Shopify App Store cạnh tranh khốc liệt, việc theo dõi đối thủ thủ công tốn quá nhiều thời gian. Các thay đổi về listing, keyword ranking, nội dung website và tài liệu hướng dẫn diễn ra liên tục — nhưng không có công cụ nào tự động phát hiện và tổng hợp tất cả những thay đổi đó.

Core Problem

Làm thế nào để một developer duy nhất có thể thiết kế kiến trúc, xây dựng và triển khai một ứng dụng SaaS full-stack cấp production? Câu trả lời: sử dụng AI agents như một đội ngũ kỹ thuật ảo — với 17 agent chuyên biệt phối hợp qua decision logs và task board.

🧠 Agentic AI — Điểm Sáng tạo Cốt lõi

Không chỉ dùng AI viết code — mà thiết kế cả một đội ngũ AI

Thay vì dùng Claude Code như trợ lý viết code đơn giản, dự án này coi AI như một đội ngũ kỹ thuật ảo. Mỗi agent có chuyên môn riêng, trách nhiệm cụ thể, quy trình vận hành chuẩn, và giao tiếp liên agent qua bảng công việc và nhật ký quyết định.

📦 Deliverables

Login Page — Deep Ocean Design System

Trang đăng nhập với Google OAuth, thiết kế theo Deep Ocean design system — dark mode, glass-morphism card với glow animation.

CEO Dashboard — Overview toàn hệ thống

Dashboard CEO hiển thị performance across all teams: stats tổng quan, team performance cards với keyword ranking badges, notable changes và recent competitor changes.

🤖 Đội ngũ 17 AI Agents

5 Agents
Development — Kiến trúc, triển khai, debug, review, refactor
6 Agents
Design — UX/UI, design system, component, page layout
5 Agents
QA — Test plan, API test, E2E test, code quality
1 Agent
Product — Yêu cầu, lộ trình, ưu tiên hóa

Hệ thống Phối hợp

  • Pre-flight Protocol: Mỗi agent kiểm tra task board và decision logs trước khi làm việc
  • Decision Logging: Ghi lại quyết định kiến trúc theo định dạng chuẩn
  • Task Board: Yêu cầu công việc liên lĩnh vực với mức ưu tiên P0-P3
  • Conflict Resolution: Agent đánh dấu xung đột, con người quyết định cuối cùng

Documentation-as-Code

  • CLAUDE.md — Tech stack và quy ước, tự động cập nhật
  • PRD.md — Tính năng, API, data models
  • Design System — Tokens, components, patterns
  • Decision Logs — 4 file cho 4 lĩnh vực (dev, UX, QA, product)

🏗️ Kiến trúc Hệ thống

Frontend

  • Next.js 15 (App Router) + React 19
  • shadcn/ui (New York) + Tailwind CSS v4
  • TanStack React Query v5 cho state management
  • NextAuth v5 beta (Google OAuth)
  • 18 trang frontend

Backend & Infrastructure

  • Express trên Firebase Cloud Functions Gen 2
  • Cloud Firestore — database
  • Puppeteer + Cheerio + @sparticuz/chromium — scraping
  • Firebase Cloud Scheduler — hàng ngày 6h sáng GMT+7
  • Telegram Bot API — thông báo thay đổi

Xác thực & Phân quyền

  • RBAC đầy đủ với 4 vai trò: CEO > PM > Contributor > Member
  • Phạm vi Team/Project: dữ liệu cách ly theo thành viên và project
  • HMAC-based server-to-server authentication

Triển khai

  • Frontend: Vercel
  • Backend: Google Cloud (Firebase) — asia-southeast1
  • Monorepo: Bun workspaces
  • Testing: Playwright E2E

⚡ Tính năng Chính

7 Scraper Tự động

  • App Listing — Theo dõi thay đổi listing trên Shopify App Store
  • Keyword Rankings — Theo dõi thứ hạng từ khóa tìm kiếm
  • Autocomplete — Giám sát gợi ý tự động của Shopify
  • Website Menus — Theo dõi menu điều hướng của đối thủ
  • Homepage Monitor — Giám sát thay đổi nội dung trang chủ
  • Guide Docs — Theo dõi thay đổi tài liệu hướng dẫn
  • Changes Feed — Tổng hợp tất cả thay đổi phát hiện được

Thành tựu Kỹ thuật

  • Dual Data Architecture — Hỗ trợ đồng thời projectId và ownerEmail, không breaking changes
  • skipIfIdenticalDiff — Giảm false-positive từ ~100% xuống ~0%
  • Deep Ocean Design System — 14 CSS tokens, dark mode first
  • Word-Level Diff — LCS algorithm tùy chỉnh, tự động fallback cho văn bản lớn
  • Serverless Pipeline — 7 scraper tuần tự trong một Cloud Function duy nhất

Key Insight

Điểm sáng tạo cốt lõi không chỉ là dùng AI để viết code, mà là thiết kế một hệ thống agentic nơi các AI agent chuyên biệt phối hợp thông qua nhật ký quyết định, bảng công việc và quy trình giải quyết xung đột — mô phỏng hiệu quả một đội ngũ kỹ thuật đầy đủ.
