Case Study 05

Joy Subscriptions — SaaS Marketing Website

Xây dựng toàn bộ marketing website cho sản phẩm SaaS Shopify bằng Claude Code — 100% AI-assisted development. Một người đảm nhận vai trò của cả team 5 người (developer, designer, copywriter, SEO specialist, brand strategist) thông qua hệ thống multi-agent AI.

📌 AI-First Development + Marketing 🏢 Joy Subscriptions 🛠️ Next.js 16 · Tailwind CSS 4 · Framer Motion · GSAP · WebGL 🚀 Live — joysubscription.com

🔍 Context & Challenge

Context

Joy Subscriptions là ứng dụng Shopify giúp merchants chuyển đổi khách hàng mua một lần thành subscribers trung thành. Sản phẩm đang phục vụ 5,000+ merchants, xử lý 10M+ subscriptions và tạo ra hơn $50M doanh thu cho merchants. Cần một marketing website chuyên nghiệp để làm conversion funnel chính.

Challenge

Xây dựng một marketing website cấp production với đầy đủ tính năng (WebGL effects, SEO infrastructure, 26 blog posts, 9 industry pages, design system) — thường cần một team 5 người. Câu hỏi: Một người + AI có thể đạt chất lượng tương đương không?

🧠 AI-First Development với Claude Code

100% AI-Assisted — Từ dòng code đầu tiên đến deployment

Toàn bộ website được xây dựng bằng Claude Code của Anthropic. Vai trò của tôi là Product Manager + Creative Director — cung cấp vision, brand direction và quality standards. Claude Code xử lý toàn bộ phần execution thông qua hệ thống multi-agent.

🤖 Hệ thống Multi-Agent

Developer
Code, kiến trúc, performance optimization
Writer

SEO Expert

Production Pipeline

Mỗi trang mới đi qua quy trình chuẩn: Content Writer soạn copy → SEO Expert tối ưu cho search → UI/UX Designer lên layout → Developer implement code → Graphic Designer đảm bảo brand consistency.

Skill Modules

  • copywriting — Brand voice guidelines, conversion-focused writing patterns
  • seo-audit — Technical SEO checklist, structured data templates
  • ui-audit — Visual consistency checks, spacing/alignment rules
  • brand-design — Color systems, typography, visual tone
  • performance-optimization — Core Web Vitals, bundle analysis
  • aio-optimization — AI Overview optimization, LLM discoverability

📦 Deliverables

Homepage — Hero & Conversion Funnel

Landing page 12 section với WebGL star field background, trust bar, feature showcase, industry solutions, testimonials và case studies. Rating 4.9/5 từ 379+ reviews.

Homepage — Full Page Overview

Tổng quan toàn bộ homepage: hero section, trust indicators, feature highlights, industry solutions grid, social proof và conversion CTAs.

Pricing Page — 3-Tier Model

Trang pricing với 3 gói (Free, Standard, Advanced), feature comparison table, FAQ section và social proof — thiết kế tối ưu cho conversion.

Blog — 26 SEO-Optimized Articles

26 bài blog SEO-optimized với category filtering, table of contents và related post suggestions — tất cả nội dung được AI nghiên cứu, viết và format.

⚡ Key Features

Design System & Architecture

  • Tailwind CSS 4 @theme tokens — zero hardcoded values
  • Modular component library (SectionHeader, FeatureCard, ContentSection, FinalCta)
  • Two-layer layout: root layout metadata + inner layout visual shell
  • Warm color palette: Deep navy (#0A014F), Cerulean (#00BCE1), Cherokee yellow (#FCD299)

Animation & Visual Effects

  • WebGL-powered galaxy star field background cho hero sections
  • Viewport-triggered Framer Motion animations (staggered reveals, fade-ups)
  • Glassmorphic card designs với gradient blobs (15-40% opacity, 80-120px blur)
  • Full accessibility: useReducedMotion(), CSS fallback via @media (scripting: none)

Performance & SEO

  • Static Site Generation (SSG) cho blog và case study pages
  • Dynamic code-splitting với lazy-loaded below-fold components
  • JSON-LD structured data (Organization, SoftwareApplication, FAQ, Article)
  • Dynamic OG image generation, canonical URLs, sitemap

Content & Conversion

  • 9 industry-specific solution pages (Coffee, Pet Food, Beauty, Supplements...)
  • 17 merchant case studies with real metrics
  • 3-tier pricing page with feature comparison and FAQ
  • Conversion-optimized CTAs across all pages

🏗️ Tech Stack

Frontend

  • Next.js 16 (App Router) + Turbopack
  • Tailwind CSS 4 với custom @theme design tokens
  • Framer Motion + GSAP + OGL (WebGL)
  • Poppins (body) + Bai Jamjuree (headings)

Infrastructure & SEO

  • SSG via generateStaticParams
  • JSON-LD schemas, dynamic OG images
  • Google Analytics (GA4) + Microsoft Clarity
  • Modular component library with barrel exports, absolute imports (@/)

Key Learning

AI la force multiplier — mot nguoi voi Claude Code co the tao ra output tuong duong mot agency nho. Nhung bottleneck khong phai la technical execution ma la vision, taste va kha nang biet can xay dung gi. Prompt engineering la ky nang moi — chat luong output phu thuoc hoan toan vao cach ban truyen dat context, constraints va creative direction.

Ve AI-Assisted Development

  • Multi-agent systems hoat dong — gan vai tro chuyen biet tao ra ket qua tot hon nhieu so voi single generic prompt
  • AI khong co nghia la chat luong thap — site co WebGL effects, accessible animations, structured data, full design system
  • Workflow matters — setup CLAUDE.md, brand guidelines, design system docs va skill modules truoc giup tiet kiem rat nhieu thoi gian

Ve vai tro con nguoi

  • Human judgment la khong the thay the — biet what to build, why it matters va whether it's good enough
  • Viet prompt tot nhu viet brief tot — can hieu san pham, doi tuong va muc tieu
  • Quality control van can nguoi that test tren thiet bi that
